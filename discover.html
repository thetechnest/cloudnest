üìù MCQs ‚Äì Check Your Understanding

1. Why was Hadoop initially chosen at Discover Card?

To store and process massive historical transaction data at low cost using distributed storage and batch processing.

2. Which Hadoop component ensures fault-tolerant storage?

HDFS with data replication.

3. Why was MapReduce not suitable for fraud detection?

Because it is disk-based and has high latency, making it unsuitable for near real-time analytics.

4. What key feature makes Spark faster than MapReduce?

In-memory processing with DAG-based execution.

5. In the hybrid architecture, which tool handled compliance jobs?

Hadoop MapReduce.
