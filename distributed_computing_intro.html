What is Distributed Computing?

Expert View ‚Äì Why Distributed Computing Emerged

Early in my career, most applications were designed to run on a single powerful server. Scaling meant buying better hardware. This model worked until applications started serving millions of users and processing enormous amounts of data.

At that point, a single machine became a bottleneck. Hardware failures caused downtime, costs increased, and performance tuning reached its limits. This forced a fundamental shift in system design ‚Äî moving from one machine to many cooperating machines.

This shift is called Distributed Computing, where multiple independent computers work together as a single system to solve large-scale problems.

üìò Exam Insight: Distributed computing emerged to overcome scalability, performance, and reliability limitations of single-machine systems.

Definition of Distributed Computing

Distributed computing is a computing model in which multiple autonomous computers communicate and coordinate over a network to achieve a common goal.

üìò Exam Line: Distributed computing involves multiple computers working together as a single system.

Why Distributed Computing is Needed

  • Scalability: Handle growing users and data
  • Performance: Parallel execution of tasks
  • Fault Tolerance: System continues despite failures
  • Cost Efficiency: Uses commodity hardware
AI/ML Context: Training large models requires distributing data and computation across machines.
üìò Exam Line: Distributed systems improve scalability, performance, and reliability.

Centralized vs Distributed Systems

Centralized System Distributed System
Single machine Multiple machines
Limited scalability Highly scalable
Single point of failure Fault tolerant
Vertical scaling Horizontal scaling

Relation to Big Data, Hadoop & Spark

Big Data systems are built on top of distributed computing principles. Hadoop distributes storage and computation, while Spark provides fast, in-memory distributed processing.

Key Insight: Big Data is an application of distributed computing at massive scale.
üìò Exam Line: Hadoop and Spark are distributed computing frameworks for Big Data processing.
