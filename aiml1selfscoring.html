Q1

Which hardware resource is most effective for accelerating deep neural network training?

Correct: GPU / TPU ‚Äî these are engineered for parallel matrix operations and dramatically reduce training time compared to CPUs.
Q2

Which managed service is an AWS offering for building, training, and deploying ML models?

Correct: Amazon SageMaker ‚Äî it provides integrated tooling for data labeling, training, tuning, deployment, and monitoring on AWS.
Q3

What does MLOps primarily aim to address?

Correct: MLOps focuses on reliable, repeatable pipelines for training, deployment, versioning, and monitoring of ML systems.
Q4

Which approach is best for scaling real-time inference with minimal operational overhead?

Correct: Serverless inference endpoints (or managed model-serving) enable autoscaling, making real-time inference easier to operate and cost-efficient for variable load.
Q5

What is the goal of distributed training?

Correct: Distributed training divides workloads across GPUs/machines to shorten total training time for large models and datasets.
Q6

Which Google Cloud product is designed for end-to-end ML workflows?

Correct: Vertex AI ‚Äî GCP's platform integrates training, pipelines, feature store, model registry and deployment capabilities.
Q7

Which tool is commonly used for containerizing ML models before deployment?

Correct: Docker containers package model code and dependencies, ensuring consistent behavior across environments.
Q8

What does a feature store provide in an ML platform?

Correct: A feature store stores computed features reliably so training and serving use consistent inputs and reduce duplication.
Q9

Which service is typically used for real-time streaming ingestion on AWS?

Correct: Kinesis is designed for ingesting, buffering and processing streaming data for real-time analytics and model inputs.
Q10

What is model versioning used for?

Correct: Versioning helps reproduce results, roll back to previous models, and compare performance across variants.
Q11

Which storage option is best for large datasets used in training (object storage)?

Correct: Object storage is cost-effective, scalable, and commonly used for storing raw datasets and model artifacts.
Q12

What is the benefit of using autoscaling for inference clusters?

Correct: Autoscaling ensures responsiveness during traffic spikes and reduces cost when demand falls.
Q13

Which orchestration platform is widely used to run containerized ML workloads at scale?

Correct: Kubernetes schedules containers across clusters, enabling resilient, scalable deployment of ML services.
Q14

Which of the following helps reduce cloud costs for training jobs?

Correct: Spot or preemptible instances are cheaper for non-time-critical training; right-sizing prevents wasted resources.
Q15

Which practice improves reproducibility of model training?

Correct: Recording configuration, seeds and dataset versions ensures training runs can be reproduced and compared.
Q16

Which security practice is important for cloud ML projects?

Correct: IAM and secrets management avoid credential leaks; least privilege reduces blast radius for breaches.
Q17

Edge deployment of ML models is best when:

Correct: Edge is used for low-latency or intermittent-connectivity scenarios; heavy training still runs in cloud data centers.
Q18

Which service is commonly used for scheduled batch ETL jobs on cloud?

Correct: Managed schedulers handle orchestration of ETL and batch pipelines with retries, logs, and dependencies.
Q19

What role do TPUs play compared to GPUs?

Correct: TPUs (Tensor Processing Units) are hardware accelerators designed by Google for matrix/tensor computations; they can outperform GPUs for some workloads.
Q20

Which component helps monitor production ML models for data drift and performance?

Correct: Observability tools capture prediction metrics, input distribution changes and trigger retraining or alerts when drift occurs.
