Cloud Computing

The Infrastructure Backbone

Cloud computing hosts the entire system ‚Äî from data storage to AI models. It enables elastic scaling during health emergencies, secure data isolation, high availability, and cost-effective deployment.

  • Runs Spark, ML models, APIs, and LLMs
  • Provides compute, storage, networking, and security

Big Data Analytics

Big Data analytics handles historical medical records and real-time sensor data. Using Hadoop and Spark, the system cleans, aggregates, and correlates data across millions of patients.

  • Kafka for real-time ingestion

Predictive Analytics

  • Risk scoring and classification models
