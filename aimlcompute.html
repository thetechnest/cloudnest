What Is Compute?

In cloud computing, the term compute refers to the processing resources required to execute software programs. This includes CPU power, memory, networking capability, and sometimes specialized hardware such as GPUs.

Applications such as machine learning models, video rendering, simulations, and large-scale web platforms are called compute-intensive applications because they demand high processing power.

Compute Resources

Compute resources are measurable units of computing power that applications request and consume dynamically from the cloud.

  • CPU ‚Äì Processing brain, measured in cores or millicores
  • Memory (RAM) ‚Äì Temporary working space for applications
  • GPU ‚Äì Specialized compute for AI/ML and graphics
  • Network ‚Äì Bandwidth and connectivity for data flow

Virtual Machines ‚Äì Core Unit of Compute

Virtual Machines (VMs) are the foundational building blocks of cloud compute services. A VM is a software-defined computer that behaves like a physical server, capable of running operating systems and applications.

Cloud platforms allow users to create, scale, and terminate VMs on-demand, without managing physical hardware.

üí° VMs provide isolation, scalability, and guaranteed compute resources.
