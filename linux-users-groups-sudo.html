Linux Users, Groups, sudo & Process Control (Zero ‚Üí Hero)

Concepts You Must Understand First

Linux is a multi-user operating system. Every command is executed by a user, governed by permissions, and runs as a process. Most Hadoop and cloud errors are caused by misunderstanding these basics.

  • User ‚Äì Who is executing a command
  • Group ‚Äì Collection of users
  • sudo ‚Äì Temporary administrative power
  • su ‚Äì Switch user
  • SSH ‚Äì Secure remote login
  • Process ‚Äì Running program
üí° Master users and permissions ‚Üí Hadoop becomes easy.

Creating Users and Groups

Create a group 
sudo addgroup hadoop

Creates a group named hadoop.

Create a user and assign group 
sudo adduser --ingroup hadoop hduser

Creates user hduser and assigns it to group hadoop.

Verify user and group 
id hduser

Displays UID, GID, and group membership.

sudo and su ‚Äì Power Control

Why sudo exists

Linux avoids permanent root usage. sudo gives admin power for a single command, reducing risk.

Make a user sudoer 
sudo usermod -aG sudo hduser

Adds hduser to sudoers group.

Test sudo access 
sudo whoami

Output: root

Switch user using su 
su - hduser

Switches session to another user.

SSH ‚Äì Secure Shell

Generate SSH key 
ssh-keygen -t rsa -P ""

Creates public/private key pair.

Enable password-less SSH 

cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys

Required for Hadoop daemons.

Test SSH 
ssh localhost

File Permissions & Ownership

Change ownership 
sudo chown hduser:hadoop /usr/local/hadoop -R

Assigns Hadoop files to correct user.

Change permissions 
chmod 755 start-dfs.sh

Read/write/execute for owner, read/execute for others.

Process Management

List processes 
ps -ef
Filter processes 
ps -ef | grep java
Kill a process 
kill <PID>

Environment Variables (.bashrc)

Edit bashrc 
nano ~/.bashrc
Set variables 

export JAVA_HOME=/usr/lib/jvm/java-11-openjdk-amd64
export HADOOP_HOME=/usr/local/hadoop/hadoop-3.3.6
export PATH=$PATH:$JAVA_HOME/bin:$HADOOP_HOME/bin
Verify using echo 
echo $JAVA_HOME
Hadoop classpath 
hadoop classpath
